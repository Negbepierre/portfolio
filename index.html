05 Security Operations

PierreGuard Ops: AI Security Operations Centre

A real-time Security Operations Centre that monitors a live AWS environment. It pulls threat findings from GuardDuty, API activity from CloudTrail, and network details from a VPC deployed using CloudFormation. Claude via AWS Bedrock analyses all of it and produces a clear threat report. The VPC uses both a Security Group and a Network ACL to block SSH, RDP, and Telnet at the subnet level.
